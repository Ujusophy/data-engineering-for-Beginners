# Module 2: Databases

Master the database fundamentals every data engineer needs to know.

## Overview

In this module, you'll learn how to work with databases - the backbone of data systems. We focus on practical skills you'll use daily as a data engineer, not deep database internals.

**Estimated Time**: 3-4 weeks (10-15 hours per week)

## Learning Objectives

By the end of this module, you will be able to:
- Set up and connect to PostgreSQL databases
- Design simple database schemas
- Understand when to use relational vs NoSQL databases
- Work with database transactions and constraints
- Connect databases to Python applications
- Perform basic database administration tasks

## Prerequisites

- Completed Module 1 (Fundamentals)
- Basic SQL knowledge
- Python fundamentals

## What You'll Need

- PostgreSQL installed locally
- Python 3.9+
- A database client (DBeaver, pgAdmin, or psql)

## Getting Started

1. [Install PostgreSQL](https://www.postgresql.org/download/)
2. Install a database client
3. Start with the concepts
4. Practice with exercises

## Key Tools

- **PostgreSQL**: Our primary database
- **psycopg2**: Python PostgreSQL adapter
- **SQLAlchemy**: Python SQL toolkit (optional)
- **DBeaver**: Universal database client
